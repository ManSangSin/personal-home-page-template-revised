User Stories

Shopping List

  1. As a user, I want to view a list of products so that I can select some to add to a list.
  2. As a user, I want to be able to see a list of items I selected so I know what I need to buy.
  3. As a user, I want to be able to remove the last item I added so that I don’t need to create a new list if I added an item by mistake.
  4. As a user, I want to be able to add different (custom) items to a list as the default items may not have what I need to buy.
  5. As a user, I want to be able to check off items which are on my list so that I know what items are bought and what still needs to be purchased.

Shopping Basket

  1. As a shopper, I want to be able to see a list of products so I can select what I want to buy.
  2. As a shopper, I want to be able to see how many of each product is in my basket so I can buy the amount I require.
  3. As a shopper, I want to be able to see the total cost so I know how much to pay.
  4. As a shopper, I want to see the price of the product so I can decide if the price is right and worth buying.
  5. As a shopper, I want to easily empty my basket if I choose I do not want to proceed with the purchase anymore.

Comment Section

  1. As a user, I want to be able to review what I write before I submit my comments so I can see if it reads correctly.
  2. As a user, I want to be able to remove my comments if I submit something by mistake.
  3. As a user, I want to be able to see other peoples comment so I know what people are saying and find out more relevent information.
